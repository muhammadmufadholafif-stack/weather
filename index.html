Tentang Aplikasi

Cloud Weather Pro adalah aplikasi cuaca modern berbasis web yang menampilkan informasi cuaca real-time dan ramalan 7 hari menggunakan Open-Meteo API.

Teknologi yang Digunakan:

HTML5 CSS3 JavaScript Open-Meteo API Geolocation API LocalStorage

Fitur Unggulan:

  • Deteksi lokasi otomatis (GPS)
  • Ramalan cuaca 7 hari ke depan
  • Data cuaca lengkap (suhu, kelembaban, tekanan, dll)
  • Waktu matahari terbit & terbenam
  • Riwayat pencarian tersimpan
  • Mode gelap & terang
  • Responsive untuk semua device
