Aplikasi

Cloud Weather adalah aplikasi web cuaca realtime yang menampilkan informasi kondisi cuaca dari berbagai kota menggunakan Public API Open-Meteo. Aplikasi ini dibuat sebagai bagian dari proyek LKS Maâ€™arif 2026 bidang Cloud Computing, memanfaatkan teknologi HTML, CSS, JavaScript, dan cloud hosting. Cloud Weather dirancang responsif untuk berbagai ukuran layar dan menghadirkan tampilan cuaca yang menarik dan mudah dipahami.

Fitur Utama

  • Input nama kota:menampilkan suhu, kecepatan angin, dan waktu update
  • Ikon cuaca sesuai kondisi (cerah, berawan, hujan, salju, badai, dll).
  • Responsif di berbagai ukuran layar
  • Data cuaca tergantung koneksi internet dan ketersediaan API, jadi jika jaringan lambat atau server API mati, aplikasi tidak bisa menampilkan cuaca.
  • Waktu update agak telat 100% langsung.
  • Tampilan masih sangat sederhana
