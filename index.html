Tentang Aplikasi

Cloud Weather adalah aplikasi web sederhana berbasis cloud computing yang menampilkan informasi cuaca realtime dari berbagai kota menggunakan Public API Open-Meteo.

berikut merupakan teknologi yang digunakan pada website ini:

HTML CSS JavaScript Public API Cloud Hosting
